# Idea-to-Social-Media Agent: HITL Content Creator

Turn any topic and style into an interleaved, multimodal storyboard ready for production publishing.

## Architecture
- **Backend:** FastAPI (Python) for rapid API routing, state management, and pipeline orchestration.
- **Frontend:** Streamlit dashboard providing a clean Human-in-the-Loop review process.
- **AI Core:** Google Gemini (`google-genai` SDK) for Meta-Prompting and Structured Outputs (Storyboard JSON). Media generation powered by the latest Gemini Image models.
- **Hosting:** Google Cloud Run for scalable, automated, and containerized deployment.

## Features
- **Human-in-the-Loop Storyboarding:** Review and approve narration, image prompts, and video prompts generated by the AI Creative Director.
- **Multimodal Generation:** Seamlessly combine text generation, image rendering, and video instruction formatting.
- **Automated Cloud Deployment:** Fully containerized environment with push-to-deploy CI/CD automation.

## CI/CD Pipeline
Every push to the `main` branch automatically triggers our GitHub Actions workflow (`.github/workflows/deploy.yml`). The workflow securely authenticates with Google Cloud, builds our custom Docker images for both the Backend and Frontend, pushes them to Google Container Registry (`gcr.io`), and finally performs a zero-downtime deployment directly to **Google Cloud Run**. The necessary API keys and references are injected securely via GitHub Secrets.

## Setup & Local Development (Docker)
The easiest way to spin up the entire application locally is using Docker Compose:

```bash
docker-compose up --build
```

This gracefully maps the FastAPI backend to `http://localhost:8000` and the Streamlit frontend to `http://localhost:8501`.
